## 🧐 What is this project?
When people search for medicine online, they often misspell names or get confused by different brands. We are building an intelligent tool that:
1. **Handles Typos:** Understands what the user meant, even with spelling errors.
2. **Matches Ingredients:** Correctly identifies medicines with the same active ingredients.
3. **Suggests Alternatives:** Finds safe replacements based on strength and dosage.

## 🎮 How to Join the Challenge
You don't need to be a senior engineer to participate! If you know basic Python and want to practice your skills, follow these steps:

1. **Fork this project** to your own GitHub account.
2. **Improve the logic** inside the `baseline_code/recommender.py` file.
3. **Submit a Pull Request** to merge your changes.

As soon as you submit, our automated "Judge" will test your code and update your rank on the leaderboard!
